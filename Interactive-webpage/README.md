# Interactive Dashboard Web Application

## Overview

This project is a simple interactive web application developed as part of my internship task. It demonstrates fundamental frontend development concepts using HTML, CSS, and JavaScript, including user authentication simulation, UI state management, local storage, and interactive features.

## Features

### User Authentication Simulation
- User login form with validation
- Username, email, and password fields
- Login success and error notifications
- User session persistence using Local Storage
- Logout functionality

### Dashboard
- Dashboard view displayed after successful login
- Login form hidden when user is authenticated
- User session maintained after page refresh

### Interactive Notifications
- Success messages for login actions
- Error messages for form validation
- Animated notification card

### Theme Toggle
- Light Mode
- Dark Mode

### Responsive Design
- Mobile-friendly layout
- Modern card-based interface
- Flexible sizing using modern CSS techniques

## Technologies Used
- HTML5
- CSS3
- JavaScript (ES6)
- Local Storage API

- UI state management
- Theme switching (Dark/Light Mode)
